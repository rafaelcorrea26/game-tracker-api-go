# 🚀 Game Tracker API

API REST para gerenciamento de jogos.

---

## 🚀 Tecnologias

- Go
- Gin
- PostgreSQL
- JWT

---

## ▶️ Rodar

```bash
go mod tidy
go run .

http://localhost:8080/swagger/index.html

Bearer Token (JWT)

📌 Endpoints

POST /register
POST /login

GET /games
POST /games
PUT /games/:id
DELETE /games/:id
